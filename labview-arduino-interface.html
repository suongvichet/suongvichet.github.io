LabVIEW-Arduino Interface

Serial communication system between LabVIEW and Arduino for controlling outputs and developing experimental engineering interfaces.

Project Overview

This project demonstrates how LabVIEW can communicate with an Arduino board through serial communication. The main purpose is to control the Arduino output from a LabVIEW graphical user interface.

Objective

  • Develop a basic LabVIEW interface to send commands to Arduino.
  • Control the Arduino built-in LED using serial commands.
  • Understand serial communication between software and hardware.
  • Prepare a foundation for future sensor acquisition and actuator control systems.

Hardware and Software

  • Arduino board
  • USB cable
  • LabVIEW software
  • Arduino IDE
  • Serial communication through COM port

Serial Command Concept

The LabVIEW interface sends simple character commands to Arduino.

  • 1 → Turn LED ON
  • 2 → Turn LED OFF

Arduino Code Concept

The Arduino continuously checks whether serial data is available. If the received character is 1, the LED turns ON. If the received character is 2, the LED turns OFF.

Result

The Arduino LED can be controlled from the LabVIEW interface by sending serial commands. This confirms that the communication between LabVIEW and Arduino is working.

Future Improvement

  • Add feedback from Arduino to LabVIEW.
  • Display sensor data in real time.
  • Control multiple LEDs, relays, or motors.
  • Use the interface for data acquisition in mechanical testing systems.
